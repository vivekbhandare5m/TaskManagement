# Task Management Application

## 📁 Project Structure

- backend/ → Spring Boot API
- frontend/ → React app 

## 🚀 Backend Setup

1. Open backend in Spring STS
2. Run the application
3. Runs on: http://localhost:8080

## 🌐 API Example

GET /api/hello

## 👨‍💻 Frontend (For React Developers)

- Create React app inside `frontend/`
- Use API base URL: http://localhost:8080

Example:
fetch("http://localhost:8080/api/...")
